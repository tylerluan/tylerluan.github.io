PR流程

  1. folk该repository

  2. clone该repository

  3. 加入远程连接

    git remote -v

    git remote add upstream https://github.com/**/*.git

  4. 建立新的分支

    git checkout -b name

  5. 修改commit 并push到新分支

  6. 发起PR “New pull requests”

  7. 等待merge
